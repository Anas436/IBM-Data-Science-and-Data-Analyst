Objectives

Hands on Lab: Watson Studio Service creation(10min)

After completing this lab, you will be able to:

  • Use Watson Studio for your data science problem solving.

Prerequisite:

You need to have an IBM Cloud Account to do this lab. If you do not have an IBM Cloud account, click and open this link and follow the instructions, to create an IBM Cloud account.

Exercise 1: Create an instance of Watson Studio service

Task 1: Add Watson Studio as a resource

  1. Go to IBM Cloud Login and login with your credentials.

  2. In the IBM Cloud Catalog search or choose AI/Machine Learning from Category.

and select Watson Studio.

  1. On the Watson Studio page, select an appropriate region depending on where you are accessing from, verify that the Lite plan is selected, and then click Create.
  1. Once the Watson Studio instance is successfully created, click on Launch IBM Cloud Pak for Data.
  1. Once you get started, the first time, your IBM Cloud Pak for Data core services is provisioned.
